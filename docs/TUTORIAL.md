## 快速开始

### 安装

1. 下载玄语编译器
2. 将编译器添加到系统路径
3. 验证安装：

```bash
xy --version
```

### 第一个程序

创建 `hello.xy` 文件：

```xy
/**
 * 我的第一个玄语程序
 */
函数 主(): 整数 {
    打印文本("你好，玄语！")
    返回 0
}
```

运行程序：

```bash
xy hello.xy --run
```

输出：
```
你好，玄语！
```

## 基础语法

### 变量定义

使用 `定义` 关键字定义变量：

```xy
// 定义不可变变量
定义 姓名: 文本 = "玄语"
定义 年龄: 整数 = 25

// 定义可变变量
定义 可变 计数: 整数 = 0
计数 = 计数 + 1  // 可以重新赋值
```

### 常量定义

使用 `常量` 关键字定义常量：

```xy
常量 圆周率: 浮点 = 3.14159
常量 最大值: 整数 = 100
```

### 注释

```xy
// 单行注释

/*
 * 多行注释
 * 第二行
 */

/**
 * 文档注释
 * @param 名称 参数说明
 * @return 返回值说明
 */
```

## 控制流

### 条件判断

```xy
定义 分数: 整数 = 85

若 分数 >= 90 则 {
    打印文本("优秀")
} 否则 若 分数 >= 60 则 {
    打印文本("及格")
} 否则 {
    打印文本("不及格")
}
```

### 循环

#### 无限循环

```xy
定义 可变 计数: 整数 = 0

循环 {
    打印整数(计数)
    计数 = 计数 + 1
    
    若 计数 >= 5 则 {
        中断  // 跳出循环
    }
}
```

#### 条件循环

```xy
定义 可变 累加: 整数 = 0

当 累加 < 10 {
    打印整数(累加)
    累加 = 累加 + 1
}
```

### 模式匹配

```xy
定义 数值: 整数 = 2

匹配 数值 {
    情况 1 则 {
        打印文本("一")
    }
    情况 2 则 {
        打印文本("二")
    }
    默认 则 {
        打印文本("其他")
    }
}
```

## 函数

### 函数定义

```xy
/**
 * 计算两数之和
 * @param 甲 第一个数
 * @param 乙 第二个数
 * @return 两数之和
 */
函数 相加(甲: 整数, 乙: 整数): 整数 {
    返回 甲 + 乙
}
```

### 函数调用

```xy
定义 结果 = 相加(3, 5)
打印整数(结果)  // 输出: 8
```

### 递归函数

```xy
/**
 * 计算阶乘
 */
函数 阶乘(数值: 整数): 整数 {
    若 数值 <= 1 则 {
        返回 1
    }
    返回 数值 * 阶乘(数值 - 1)
}

打印整数(阶乘(5))  // 输出: 120
```

### 泛型函数

```xy
/**
 * 交换两个值
 */
函数 交换<T>(甲: T, 乙: T): (T, T) {
    返回 (乙, 甲)
}
```

## 结构体与枚举

### 结构体

```xy
/**
 * 定义结构体
 */
结构体 用户 {
    姓名: 文本,
    年龄: 整数
}

/**
 * 使用结构体
 */
函数 主(): 整数 {
    定义 用户实例 = 用户 {
        姓名: "玄语",
        年龄: 1
    }
    
    打印文本(用户实例.姓名)
    打印整数(用户实例.年龄)
    
    返回 0
}
```

### 枚举

```xy
/**
 * 定义枚举
 */
枚举 颜色 {
    红,
    绿,
    蓝
}

/**
 * 使用枚举
 */
函数 主(): 整数 {
    定义 当前颜色 = 颜色.红
    
    匹配 当前颜色 {
        情况 颜色.红 则 {
            打印文本("红色")
        }
        情况 颜色.绿 则 {
            打印文本("绿色")
        }
        情况 颜色.蓝 则 {
            打印文本("蓝色")
        }
    }
    
    返回 0
}
```

## 进阶特性

### 闭包

```xy
函数 创建计数器(): () -> 整数 {
    定义 可变 计数: 整数 = 0
    
    返回 () => {
        计数 = 计数 + 1
        返回 计数
    }
}

定义 计数器 = 创建计数器()
打印整数(计数器())  // 输出: 1
打印整数(计数器())  // 输出: 2
```

### 尾递归优化

玄语编译器会自动优化尾递归调用：

```xy
函数 尾递归求和(数值: 整数, 累加: 整数): 整数 {
    若 数值 == 0 则 {
        返回 累加
    }
    返回 尾递归求和(数值 - 1, 累加 + 数值)
}
```

### 外部函数

调用 C 语言库函数：

```xy
函数 系统时间(): 整数 {
    外部 "time"
}
```

## 最佳实践

### 命名规范

- 使用有意义的中文命名
- 函数名使用动词短语
- 变量名使用名词
- 常量使用全大写

### 代码风格

- 使用 4 空格缩进
- 每行不超过 80 字符
- 函数之间空一行
- 添加必要的注释

### 错误处理

```xy
函数 安全除法(甲: 整数, 乙: 整数): 整数 {
    若 乙 == 0 则 {
        打印文本("错误：除数不能为零")
        返回 0
    }
    返回 甲 / 乙
}
```
