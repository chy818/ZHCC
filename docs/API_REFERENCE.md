## 内置类型

### 基本类型

| 类型 | 中文名 | 描述 | 示例 |
|------|--------|------|------|
| `Int` | 整数 | 32位有符号整数 | `42`, `-10`, `0` |
| `Long` | 长整数 | 64位有符号整数 | `9223372036854775807L` |
| `Float` | 浮点 | 单精度浮点数 | `3.14`, `-0.5` |
| `Double` | 双精度 | 双精度浮点数 | `3.14159265358979` |
| `Bool` | 布尔 | 布尔值 | `真`, `假`, `true`, `false` |
| `String` | 文本 | UTF-8 编码字符串 | `"你好"`, `"Hello"` |
| `Void` | 空类型 | 无返回值 | 用于函数返回类型 |

### 复合类型

| 类型 | 语法 | 描述 |
|------|------|------|
| 数组 | `T[]` | 元素类型为 T 的数组 |
| 列表 | `List<T>` | 动态列表 |
| 可选 | `Optional<T>` | 可能为空的值 |
| 函数 | `(T1, T2) -> R` | 函数类型 |
| 元组 | `(T1, T2)` | 固定长度的异构集合 |

## 内置函数

### 输入输出

#### `打印整数(数值: 整数)`

打印整数到标准输出。

**参数:**
- `数值`: 要打印的整数

**示例:**
```xy
打印整数(42)  // 输出: 42
```

---

#### `打印浮点数(数值: 浮点)`

打印浮点数到标准输出。

**参数:**
- `数值`: 要打印的浮点数

**示例:**
```xy
打印浮点数(3.14)  // 输出: 3.14
```

---

#### `打印文本(文本: 文本)`

打印文本到标准输出。

**参数:**
- `文本`: 要打印的字符串

**示例:**
```xy
打印文本("你好，玄语！")  // 输出: 你好，玄语！
```

---

#### `读取整数(): 整数`

从标准输入读取一个整数。

**返回值:**
- 读取的整数

**示例:**
```xy
定义 数值 = 读取整数()
打印整数(数值)
```

## 标准库

### string 模块

#### `字符串长度(文本: 文本): 整数`

计算字符串长度。

**参数:**
- `文本`: 输入字符串

**返回值:**
- 字符串长度

---

#### `字符串拼接(甲: 文本, 乙: 文本): 文本`

拼接两个字符串。

**参数:**
- `甲`: 第一个字符串
- `乙`: 第二个字符串

**返回值:**
- 拼接后的字符串

---

### array 模块

#### `数组长度<T>(数组: T[]): 整数`

获取数组长度。

**参数:**
- `数组`: 输入数组

**返回值:**
- 数组长度

---

#### `数组求和(数组: 整数[]): 整数`

计算整数数组元素之和。

**参数:**
- `数组`: 整数数组

**返回值:**
- 元素之和

---

### math 模块

#### `绝对值(数值: 整数): 整数`

计算整数的绝对值。

**参数:**
- `数值`: 输入整数

**返回值:**
- 绝对值

---

#### `平方根(数值: 浮点): 浮点`

计算平方根。

**参数:**
- `数值`: 输入浮点数

**返回值:**
- 平方根

---

#### `阶乘(数值: 整数): 整数`

计算阶乘。

**参数:**
- `数值`: 非负整数

**返回值:**
- n!

---

#### `斐波那契(数值: 整数): 整数`

计算斐波那契数。

**参数:**
- `数值`: 索引（从 0 开始）

**返回值:**
- 第 n 个斐波那契数
