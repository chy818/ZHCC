# 玄语标准库 (XY Standard Library)

玄语标准库提供了常用的数据结构和工具函数，帮助开发者快速构建应用程序。

## 模块列表

### 📝 string - 字符串处理

提供字符串操作的工具函数。

| 函数 | 描述 |
|------|------|
| `字符串长度(文本)` | 计算字符串长度 |
| `字符串拼接(甲, 乙)` | 拼接两个字符串 |
| `字符串比较(甲, 乙)` | 比较两个字符串 |
| `字符串复制(目标, 源)` | 复制字符串 |
| `查找字符(文本, 字符)` | 查找字符位置 |
| `字符串转整数(文本)` | 字符串转整数 |
| `整数转字符串(数值)` | 整数转字符串 |
| `字符串为空(文本)` | 检查字符串是否为空 |
| `以...开头(文本, 前缀)` | 检查是否以指定前缀开头 |

### 📊 array - 数组操作

提供数组操作的工具函数。

| 函数 | 描述 |
|------|------|
| `数组长度(数组)` | 获取数组长度 |
| `数组求和(数组)` | 计算数组元素之和 |
| `数组最大值(数组)` | 获取数组最大值 |
| `数组最小值(数组)` | 获取数组最小值 |
| `数组平均值(数组)` | 计算数组平均值 |
| `数组反转(数组)` | 反转数组 |
| `数组包含(数组, 元素)` | 检查数组是否包含元素 |
| `数组查找索引(数组, 元素)` | 查找元素索引 |

## 使用示例

```xy
// 导入标准库
导入 "stdlib/std.xy"

函数 主(): 整数 {
    // 字符串操作
    定义 文本 = "你好，玄语！"
    打印文本(文本)
    打印整数(字符串长度(文本))
    
    // 数组操作
    定义 数值列表 = [1, 2, 3, 4, 5]
    打印整数(数组求和(数值列表))
    打印整数(数组最大值(数值列表))
    
    // 数学运算
    打印整数(绝对值(-10))
    打印浮点数(平方根(16.0))
    打印整数(阶乘(5))
    
    返回 0
}
```
