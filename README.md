# 玄语编译器 (XuanYu Compiler)

> 🚧 **项目状态**：Work In Progress (Alpha)

> 本项目原名 ZHCC，现正式更名为「玄语」。代码处于早期阶段，架构可能随时调整。欢迎提 Issue 或 PR 共同完善！

## ✨ 特性

- 🔤 **中文关键字**：使用中文关键字进行编程，如 `函数`、`定义`、`若`、`循环` 等
- ␣ **语义空格**：关键字后建议添加空格（可选警告），提升代码可读性
- 🛡️ **内存安全**：支持所有权和借用检查
- 🎯 **RISC-V 目标**：编译生成 RISC-V RV64GC 架构的代码
- 📜 **LLVM IR 生成**：生成可读的 LLVM 中间表示

## 快速开始

```bash
# Debug 构建
cargo build

# 运行示例
./target/debug/xy.exe examples/hello.xy
```

### 示例代码

```xy
// Hello World 示例
函数 主 函数(): 整数 {
    定义 消息: 文本 = "你好, 世界!"
    打印(消息)

    // 条件语句
    定义 分数: 整数 = 85
    若 分数 > 60 则 {
        打印("及格")
    } 否则 {
        打印("不及格")
    }

    // 循环
    定义 i: 整数 = 0
    当 i < 10 则 {
        i = i + 1
    }

    返回 0
}
```
