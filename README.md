# zhcc - 中文计算体系编译器

基于 CCAS-Spec-v2.0 规范实现的中文编程语言编译器，支持 RISC-V RV64GC 架构。

## 特性

- **中文关键字**：使用中文关键字进行编程，如 `函数`、`定义`、`若`、`循环` 等
- **语义空格**：关键字后必须添加空格，提升代码可读性
- **内存安全**：支持所有权和借用检查
- **RISC-V 目标**：编译生成 RISC-V RV64GC 架构的代码

## 关键字

### 控制流
| 关键字 | 说明 |
|--------|------|
| `若` / `则` / `否则` / `否则若` | 条件语句 |
| `当` / `直到` | while 循环 |
| `循环` / `从` / `到` / `取自` | for 循环 |
| `匹配` / `情况` / `默认` | match 语句 |

### 函数
| 关键字 | 说明 |
|--------|------|
| `函数` | 函数定义 |
| `返回` | 返回值 |
| `异步` / `等待` / `启动` | 异步编程 |

### 类型
| 关键字 | 说明 |
|--------|------|
| `整数` / `长整数` | 整数类型 |
| `浮点数` / `双精度` | 浮点数类型 |
| `布尔` | 布尔类型 |
| `文本` / `字符` | 字符串和字符 |
| `无返回` | void 类型 |

### 内存安全
| 关键字 | 说明 |
|--------|------|
| `定义` | 变量声明 |
| `可变` | 可变绑定 |
| `借用` / `可变借用` | 借用检查 |
| `拥有` | 所有权 |
| `原生` | unsafe 代码 |

## 构建

```bash
# Debug 构建
cargo build

cargo build --release

# 运行测试

```

## 使用

```bash
# 编译源文件
./target/release/zhcc.exe examples/hello.zh

# 查看帮助
./target/release/zhcc.exe --help
```

## 示例代码

```zh
// Hello World 示例
函数 主 函数() {
    定义 消息: 文本 = "你好, 世界!"
    打印(消息)
    
    // 条件语句
    定义 分数: 整数 = 85
    若 分数 > 60 则 {
        打印("及格")
    } 否则 {
        打印("不及格")
    }
    
    // 循环
    定义 i: 整数 = 0
    当 i < 10 则 {
        i = i + 1
    }
    
    返回 0
}
```
