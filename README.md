# 玄语编译器 (XuanYu Compiler)

> 🚧 **项目状态**：Work In Progress (Alpha) 

> 本项目原名 ZHCC，现正式更名为「玄语」。代码处于早期阶段，架构可能随时调整。欢迎提 Issue 或 PR 共同完善！

## ✨ 特性

- 🔤 **中文原生语法**：使用中文关键字编程，如 `函数`、`定义`、`若`、`循环` 等
- ␣ **语义空格**：关键字后建议添加空格（可选警告），提升代码可读性
- 🛡️ **内存安全**：编译期所有权和借用检查，无 GC 也安全
- 🎯 **多架构支持**：基于 LLVM，支持 RISC-V RV64GC、x86\_64、ARM 等
- 📜 **LLVM IR**：生成可读的 LLVM 中间表示
- 🔌 **C/C++ 互操作**：通过 FFI 直接调用现有生态库

## 快速开始

### 环境要求

- Rust 1.70+
- LLVM 15+ (用于代码生成)
- Clang (用于链接)

### 构建与运行

```bash
# Debug 构建
cargo build

# Release 构建 (优化)
cargo build --release

# 编译并运行示例
cargo run --example hello
# 或
cargo run -- --run examples/hello.xy
```

### 命令行选项

```bash
xy <源文件> [选项]

选项:
  -h, --help    显示帮助信息
  --ir          只生成 LLVM IR (默认)
  --build       生成可执行文件
  --run         编译并运行程序
```

### 示例代码

```xy
// Hello World 示例
函数 主 函数(): 整数 {
    定义 消息: 文本 = "你好, 世界!"
    打印(消息)

    // 条件语句
    定义 分数: 整数 = 85
    若 分数 > 60 则 {
        打印("及格")
    } 否则 {
        打印("不及格")
    }

    // 循环
    定义 i: 整数 = 0
    当 i < 10 则 {
        i = i + 1
    }

    返回 0
}
```

### 数据结构

```xy
// 结构体定义
结构体 Token {
    名称: 文本,
    内容: 文本,
    行号: 整数,
    列号: 整数
}

// 使用
定义 token: Token = Token {
    名称: "标识符",
    内容: "foo",
    行号: 1,
    列号: 0
}

// 字段访问
打印(token.名称)
```


```
│   │   └── ast.rs       # AST 节点定义

│   ├── types/           # 类型系统
│   │   └── types.rs     # 类型定义
│   ├── error/           # 错误处理
│   │   └── error.rs     # 错误类型定义
│   ├── lib.rs           # 库入口
│   └── main.rs          # 命令行入口
├── runtime/             # 运行时库
│   └── runtime.c        # C 运行时实现
```

## 技术架构

```
源代码 (.xy)
     │
     ▼
  ┌──────────────────┐
  │   词法分析器      │  Token 流
  │   (Lexer)        │──────────┐
  └──────────────────┘          │
                                ▼
  ┌──────────────────┐    AST 抽象语法树
  │   语法分析器      │◀─────────┤
  │   (Parser)       │          │
  └──────────────────┘          │
                                ▼
  ┌──────────────────┐    语义分析
  │   语义分析器      │──────────┤
  │   (Sema)         │          │
  └──────────────────┘          │
                                ▼
  ┌──────────────────┐    LLVM IR
  │   代码生成器      │◀─────────┤
  │   (Codegen)      │          │
  └──────────────────┘          │
                                ▼
     可执行文件 / LLVM IR / 目标文件
```
